What is Cobweb Algorithm

It is an unsupervised, incremental, conceptual and polythetic clustering algorithm.

  • Unsupervised
  • Incremental
  • Conceptual
  • Polythetic

Unsupervised

When we talk about unsupervised algorithm, we are actually talking about one of the two important learning algorithms in machine learning .The other one is supervised algorithm. To decide a problem is unsupervised or not is based on whether we know the desired outcome for the input data. In unsupervised learning, we donâ€™t know the correct answer for each example. What we need to do is to let the algorithm discover any interesting structure in the data.

Incremental

It means that data is inputted as a stream. Only one item is processed at a time.

Conceptual

What makes conceptual clustering differ from ordinary data clustering is that a concept description will be created for each cluster generated as the algorithm proceeds.

Polythetic

It means COBWEB considers all the attribute values when an new item is added instead of evaluate each attribute value one by one.
